# Robot cuadrúpedo para detección de gases

Este repositorio contiene el código utilizado para el control de un robot cuadrúpedo orientado a la detección de gases combustibles en espacios industriales confinados.

## Componentes principales
- ESP32-CAM
- Sensor MQ-4
- Controlador PCA9685
- Servomotores SG90/MG90S
- Comunicación inalámbrica WiFi

## Funciones
- Control de locomoción
- Transmisión de video
- Lectura del sensor MQ-4
- Control remoto inalámbrico
